# AZ-900: Microsoft Azure Fundamentals — Exam Skills Index

> Outline current as of July 20, 2026 (per Microsoft Learn). Exam skills, weightings, and format are revised periodically — check the [official exam page](https://learn.microsoft.com/en-us/credentials/certifications/exams/az-900/) for the latest before you sit the exam.

This file maps the official "skills measured" outline to the modules in this repo, so you can jump straight to the material for any exam objective. Use it as your starting point and checklist.

---

## Exam basics

| | |
|---|---|
| **Passing score** | 700 (out of 1000) |
| **Domains** | Describe cloud concepts (25–30%) · Describe Azure architecture and services (35–40%) · Describe Azure management and governance (30–35%) |
| **Question scope** | Mostly general availability (GA) features; Preview features only if commonly used |
| **Accommodations** | Available on request for assistive devices, extra time, or other modifications |
| **Practice** | A free Practice Assessment is available via Microsoft Learn before booking the real exam |

Localized exam versions typically update ~8 weeks after the English version; if your preferred language isn't yet updated, you can request an extra 30 minutes.
